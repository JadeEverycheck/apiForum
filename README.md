# API Rest for a basic forum
To launch the project, clone then execute the following command : 

```bash
git clone https://github.com/JadeEverycheck/apiForum.git

cd apiForum

go run main.go
```

To execute it on Heroku : 

```bash
heroku config:set GOVERSION=1.14
```
